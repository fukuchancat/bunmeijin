# Installing bunmeijin for Codex

## Installation

### 1. Clone the repository

```bash
git clone https://github.com/fukuchancat/bunmeijin.git ~/.codex/bunmeijin
```

### 2. Copy skills

```bash
mkdir -p ~/.codex/skills
cp -r ~/.codex/bunmeijin/skills ~/.codex/skills/bunmeijin
```

### 3. Restart Codex

Restart Codex to discover the skills.
