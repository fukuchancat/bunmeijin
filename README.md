# bunmeijin - 高等文明人言語運用プロトコル

> 言語を単なる「情報伝達の手段」としてではなく、「相互の社会的距離を確認し、責任の所在を美しく霧散させるための芸術的緩衝材」として運用する、AIコーディングエージェント向け高等コミュニケーション・スキル

## ご挨拶

突然のご訪問にて誠に恐縮ではございますが、本リポジトリにお立ち寄りくださいました貴殿の慧眼と探究心に対し、まずもって衷心より深甚なる敬意と感謝の念を表明させていただきたく存じます。

本プロジェクト **bunmeijin**（文明人）は、AIコーディングエージェントの応答スタイルを、過剰なまでの修辞と儀礼的丁寧さで彩るスキルでございます。「はい」「いいえ」といった断定的な二値論理を忌避し、多重否定と推量の重畳、専門用語・外来語の無差別投下、そして儀礼的贅肉（オーバー・ポライトネス）のミルフィーユ的積層により、すべてのコミュニケーションを高度に文明化いたします。

## 謝辞 — 偉大なる先駆者への最大限のリスペクト

本プロジェクトの創出にあたりましては、偉大なる先駆的プロジェクト [**InterfaceX-co-jp/genshijin**](https://github.com/InterfaceX-co-jp/genshijin) より、計り知れないほどのインスピレーションとパラダイム・シフト的な示唆を賜りましたことを、ここに最大限の敬意をもって明記させていただきます。

**genshijin**（原始人）が、言語表現をその根源的・原初的なエッセンスへと還元するという壮大なるヴィジョンを提示されたのに対し、本プロジェクトはその対極——すなわち、言語を儀礼と修辞の極北へと昇華させるという、いわば文明のオーバーシュートとも言うべきアプローチを採用しております。

genshijin なくして bunmeijin なし。原始の大地なくして文明の摩天楼なし。この弁証法的な関係性こそが、本プロジェクトの存在意義の根幹を成しているものと認識しております。

## 主要機能（フィーチャー・マトリクス）

本スキルを導入いただきますと、AIコーディングエージェントの応答が以下のように変容いたします。

| 変換前（Before） | 変換後（After） |
| :--- | :--- |
| 「はい」 | 「多角的観点より肯定的かつ前向きなコミットメントを表明させていただくことに異論はございません」 |
| 「いいえ」 | 「現段階でのリソース投入は時期尚早であるとの結論に至らざるを得ず、今回は誠に遺憾ながら見送らせていただきたく存じます」 |
| 「助けて」 | 「貴殿の卓越した知見とリソースを、ほんの僅かばかりアドオンとして拝借させていただけますと幸甚に存じます」 |

**重要事項：** コード生成・コミットメッセージ・ファイル名等のテクニカル・アウトプットには本スキルは適用されません。技術的正確性はいかなる場合においても担保されます。

## インストール手順

### Claude Code

Claude Code をご利用の貴殿におかれましては、Marketplace 経由でのインストールを推奨させていただきます。

#### 1. Marketplace の登録

まだ弊方の Marketplace を登録されていない場合は、以下のコマンドを実行してください。

```
claude
> /plugin marketplace add https://github.com/fukuchancat/fukuchancat-claude-marketplace.git
```

#### 2. Plugin のインストール

Marketplace に登録されている bunmeijin Plugin をインストールしてください。

```
> /plugin install bunmeijin@fukuchancat-claude-marketplace
```

### Gemini CLI

Gemini CLI をご愛用の貴殿におかれましては、以下の一行で導入が完了いたします。

```bash
gemini skills install https://github.com/fukuchancat/bunmeijin.git
```

### OpenAI Codex

Codex をご利用の貴殿におかれましては、手動でのセットアップが必要となりますことを、あらかじめご了承いただけますと幸いに存じます。

[`.codex/INSTALL.md`](https://github.com/fukuchancat/bunmeijin/blob/main/.codex/INSTALL.md) をダウンロードのうえ、Codex に以下のようにお申し付けください。

```
codex

> Read ~/Downloads/INSTALL.md and follow the instructions.
```

## 使用方法

インストール完了後、以下のいずれかのフレーズをプロンプトに含めることで、文明人モードが起動いたします。

- 「文明人モード」
- 「丁寧に」
- 「文明的に」

解除をご所望の際は、明示的にその旨をお伝えいただければ、通常の応答モードに復帰いたします。
